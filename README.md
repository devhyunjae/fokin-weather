# fokin-weather
## How to play
  1. Download Expo
    For Android, 
    https://play.google.com/store/apps/details?id=host.exp.exponent
    For iOS, 
    https://apps.apple.com/kr/app/expo-client/id982107779
  2. Click link
    https://expo.io/@devhyunjae/fokin-weather
  3. Click `Open project using Expo`
